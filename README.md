# PersianStemmer-Java

## Build

```console
gradlew clean
gradlew build
```

## Code Example

```java
    import com.intellitext.persian.Stemmer;
    
    Stemmer ps = new Stemmer();
    System.out.println(ps.run("زیباست"));
```
