# PersianStemmer-Java

Go to dist folder to use PersianStemmer!

## Code Example

```java
    import htz.ir.stemming.PersianStemmer;
    
    PersianStemmer ps = new PersianStemmer();
    System.out.println(ps.run("زیباست"));
```
